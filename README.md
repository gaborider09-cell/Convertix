# proyecto-qvrd0c

Este ZIP contiene tu proyecto completo (HTML + assets como GLB, imágenes, etc.).

## Cómo abrirlo
1. Descomprime el ZIP en una carpeta.
2. Abre `index.html` en un servidor local (los navegadores bloquean cargar GLB por `file://`).

   Opción rápida con Python:
   ```
   python -m http.server 8080
   ```
   y luego abre http://localhost:8080

También puedes subirlo a Netlify, Vercel, GitHub Pages o CodeSandbox tal cual.
